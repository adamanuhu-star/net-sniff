# 📡 NET-SNIFF

A browser-based packet sniffer and network traffic visualizer that simulates TCP/IP packet capture and analysis.

## 🚀 Live Demo

https://adamanuhu-star.github.io/net-sniff

## 📊 Features

- **Real-time Packet Capture** — Simulated network traffic monitoring
- **Protocol Analysis** — TCP, UDP, ICMP, HTTP header inspection
- **Packet Inspector** — Deep dive into individual packet fields
- **Hex Dump Viewer** — Raw packet data in hexadecimal format
- **TCP/IP Stack Visualization** — Interactive OSI model layer highlighting
- **Traffic Statistics** — Bytes captured, packet rates, top talkers
- **Protocol Filtering** — Filter by TCP, UDP, ICMP, or HTTP

## 🛠️ Tech Stack

- HTML5, Tailwind CSS, Vanilla JavaScript
- No external dependencies
- Simulated packet generation for demo purposes

## 🎯 Networking Concepts Demonstrated

- **TCP/IP Stack:** Understanding of OSI layers and encapsulation
- **Protocol Headers:** Source/dest IPs, ports, flags, checksums, TTL
- **Packet Analysis:** Hex dump interpretation and field extraction
- **Traffic Patterns:** Protocol distribution and network behavior
